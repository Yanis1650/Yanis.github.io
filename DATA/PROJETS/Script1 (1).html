Assemblage des fichiers annuels DVF en une seule archive - France entière

Utilisation du package data.table

Ce package permet un assemblage rapide de tableaux. On intègre dans une nouvelle table vide data.table que l’on renomme FranceEntiere. Avec la fonction files_names <- list.files(pattern = “full”), on choisit dans notre répertoire tous les fichiers contenant l’argument full dans leur intitulé.

Pour la suite, nous créons une boucle. Nous avons une itération sur chaque fichier CSV, avec un filtre sur les types de mutations, types maison et/ou appartement, et nous renommons chaque colonne. La fonction rbind permet l’empilement des fichiers.

start.time <- Sys.time()

# Créez un data.table vide pour l'assemblage dans lequel seront assemblés les fichiers importés 
FranceEntiere <- data.table()

# pour créer une liste avec le nom des fichiers csv contenus dans le répertoire de travail, pattern = "full" permet de récupérer tous les fichiers contenants full = fichier de mutations
files_names <- list.files(pattern = "full")

# pour créer un objet contenant le nombre de fichiers csv
nb_files <- length(files_names) 

# démarrage de l'import et de la sélection des données par année
for (i in 1:nb_files) {
  temp <- fread(files_names[i], encoding = "UTF-8")  # Utilisez fread pour lire les fichiers CSV
  
  # Filtrez et sélectionnez les données
  temp <- temp[nature_mutation == "Vente" & (type_local == "Maison" | type_local == "Appartement")]
  
  # Sélectionnez les colonnes
  temp <- temp[, .(id = id_mutation, 
                   disposition = numero_disposition, 
                   parcelle = id_parcelle, 
                   date = date_mutation,
                   nature = nature_mutation, 
                   codecommune = code_commune, 
                   departement = code_departement,
                   type = type_local, 
                   surface = surface_reelle_bati, 
                   piece = nombre_pieces_principales,
                   prix = valeur_fonciere, latitude, longitude)]
  
  # Remplacez les cellules vides par des NA
  temp[temp == ""] <- NA
  
  # Empilez les fichiers annuels
  FranceEntiere <- rbindlist(list(FranceEntiere, temp))
}
